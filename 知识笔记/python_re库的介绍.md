# re的介绍

re不是第三方库，而是python的标准库（内置模块），无需额外安装，直接通过 import re 即可使用457。它提供完整的正则表达式支持，用于字符串的模式匹配、搜索、替换等操作。



## 📚 一、核心功能与常用函数
`re` 库的核心功能围绕正则表达式的匹配、提取、替换展开，以下是其核心函数及用途：

| 函数             | 作用描述                                                                 | 示例                                                                 |
|------------------|--------------------------------------------------------------------------|----------------------------------------------------------------------|
| `re.match()`     | **从字符串开头匹配**模式，成功返回 `Match` 对象，否则返回 `None`         | `re.match(r'\d+', '123abc')` → 匹配 `'123'`                         |
| `re.search()`    | **扫描整个字符串**，返回第一个匹配的 `Match` 对象                        | `re.search(r'\d+', 'abc123')` → 匹配 `'123'`                        |
| `re.findall()`   | 返回所有**非重叠匹配**的列表（字符串或元组）                             | `re.findall(r'\d+', 'a1b22')` → `['1', '22']`                       |
| `re.finditer()`  | 返回所有匹配的**迭代器**（每个元素是 `Match` 对象），适合大数据处理       | `[m.group() for m in re.finditer(r'\d+', 'a1b22')]` → `['1', '22']` |
| `re.sub()`       | **替换匹配项**，返回新字符串                                             | `re.sub(r'\d+', '#', 'a1b22')` → `'a#b#'`                           |
| `re.split()`     | **按模式分割字符串**，返回列表                                           | `re.split(r'\W+', 'Hello, world!')` → `['Hello', 'world', '']`      |

---
## 🔍 二、匹配对象（`Match`）与分组
匹配成功返回的 `Match` 对象提供以下方法提取信息：

- **`group(n)`**：返回第 `n` 个分组（`0` 表示整个匹配）
- **`groups()`**：返回所有分组的元组
- **`start()`/`end()`**：返回匹配的起止索引
- **`span()`**：返回匹配范围 `(start, end)`

**示例**：
```python
text = "Date: 2023-10-05"
match = re.search(r'(\d{4})-(\d{2})-(\d{2})', text)
match.group(1)    # '2023'
match.groups()    # ('2023', '10', '05')
match.span(1)     # (6, 10)
```
